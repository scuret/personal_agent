This is a single-user, local-first AI agent. It stores a lot of personal data on your machine and sends a lot of personal data to Anthropic on your behalf. Worth understanding what you're signing up for before you configure it.

  • Costs real money. Each Claude turn is $0.01–0.05; the morning brief is $0.10–0.30. Typical day: $1–3. Build days spike higher. Watch spend at /observability.
  • Stores everything locally. Every conversation, every fact, every Claude API payload lives in data/memory.sqlite (gitignored, owner-only file perms).
  • Web UI has no authentication. Binds to 127.0.0.1 only. Treat it like a local database port — don't expose to LAN.
  • Sends real personal email content to Anthropic. The scheduler's email triage reads every non-automated unread Gmail message via a Haiku call. Set EMAIL_TRIAGE_LOCAL_ONLY=true later to opt out.

Full details at /about/privacy.
